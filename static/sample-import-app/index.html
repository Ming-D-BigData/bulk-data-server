Introduction

This app demonstrates a basic bulk import of FHIR resources, based on the $import operation proposal. Command-line equivalents of the generated request are made available at the bottom of the page for reference.

Metadata about Import

URI to indicate provenance of data to be imported. May be used for tracking this set of imported data throughout its lifecycle. May be a FHIR endpoint that the importing system could use when matching references to previously imported data

Default storage source is simply "https" with vendor-specific implementations possible.
